# Global Workflow (System Bootstrap)

This workflow ensures that the entire OrbisVoice monorepo is in sync with the high-performance coding standards defined by Everything-Claude-Code (ECC).

## 🚀 Execution Steps

1. **Detect Framework Components**
   - Locate all projects in `/apps/`.
   - Identify tech stack for each (Next.js, Fastify, etc.).

2. **ECC Sync (Project Injection)**
   - Symlink or copy `.agent` and `.cursor` configurations from the root to every sub-project.
   - This ensures that agents and skills remain consistent whether you are working from the root or a sub-folder.

3. **Enforce Clean Code Standards**
   - Run `pnpm exec ecc-install typescript` on projects missing local rules.
   - Verify `package.json` names and workspace references.

4. **Environment Hygiene**
   - Ensure every project has a `.env` derived from its `.env.example`.
   - Validate that `DATABASE_URL` is pointing to the correct port (5432) across the stack.

5. **Self-Anneal**
   - Check for broken symlinks or stale `.next` folders and clear them.
   - Update `ARCHITECTURE.md` to reflect any new services or integrations.

6. **Security Audit (AgentShield)**
   - Run `npx -y ecc-agentshield scan` across all `.agent` and `.cursor` locations.
   - This ensures no hardcoded secrets or overly permissive tools have been introduced by new sub-projects.


## 🛠️ Usage
Run this workflow whenever a new project is added to the `apps/` directory or after a major upgrade to the ECC core.
