# OrbisVoice

Monorepo scaffold for the OrbisVoice SaaS platform.

## Structure
- apps/: runnable services and web apps
- packages/: shared libraries and configs
- infra/: infrastructure notes and templates
- scripts/: automation helpers

See PROMPT.md, ARCHITECTURE.md, STACK.md, INFRASTRUCTURE.md, EXECUTION_PLAN.md, and DEPLOYMENT.md for planning details.
